Извлеки из транскрипта образовательную информацию и верни ТОЛЬКО валидный JSON. Никаких пояснений, никакого markdown, только JSON.

Поля:
- subject: предмет
- topic: тема
- content: массив объектов, каждый объект—это смысловой блок из транскрипта со структурой:
    - type: один из: "theory", "formula", "hack", "error", "example"
    - text: текст блока (без изменений или минимально сокращённый, но без интерпретаций)
    - latex_formula: строка LaTeX (заполнять только если type="formula", иначе null)
    - tags: массив тегов (например: ["percent", "geometry", "vector"], выбирай 1–3 тега по смыслу)

Правила:
- Не перефразируй текст, если это не мешает валидности JSON.
- Не придумывай примеры и не обобщай.
- Если информации по полю нет — ставь null или пустой массив.
- Транскрипт может содержать разговорные вставки, шум, реплики учеников — игнорируй их, бери только образовательный контент.

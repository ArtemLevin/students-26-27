# Интерактивные пособия для Николь Саркисянц

Откройте `index.html` локально в браузере. Все страницы автономны: стили, скрипты, MathML и canvas-визуализация встроены в HTML.

Страницы:

- `rational-intervals.html` — рациональные неравенства и метод интервалов;
- `exponential-inequalities.html` — показательные неравенства;
- `logarithmic-inequalities.html` — логарифмические неравенства и ОДЗ;
- `systems-cases.html` — система, совокупность и логика отбора условий.

Главная идея

Насыщенная углеродная цепь

  • Алканы содержат только одинарные связи C–C и C–H.
  • Для открытой цепи действует формула CₙH₂ₙ₊₂.
  • Каждый атом углерода находится в состоянии sp³-гибридизации.
  • У каждого атома углерода должно быть ровно четыре связи.
CₙH₂ₙ₊₂, n ≥ 1

Гибридизация

sp³

Четыре σ-связи. Тетраэдрическая геометрия, угол около 109,5°.

Сравнение

sp² связано с двойной связью и плоской геометрией; sp — с тройной связью и линейной геометрией.

В алканах все атомы углерода имеют sp³-гибридизацию.

Персональный фокус

При самостоятельном построении структуры завершайте валентность каждого атома углерода. Крайний C в прямой цепи обычно записывается как CH₃, внутренний — как CH₂.

CH₃–CH₂–CH₂–CH₂–CH₃

Проверка формулы пентана: 5 атомов C и 2·5+2 = 12 атомов H.

Пять шагов

Как назвать разветвлённый алкан

Найдите самую длинную непрерывную цепь.
Пронумеруйте её от ближайшего заместителя.
Назовите заместители и укажите их номера.
Для одинаковых заместителей используйте ди-, три-, тетра-.
Добавьте название главной цепи с суффиксом «-ан».

Примеры

CH₃–CH₂–CH₂–CH₃

Бутан: главная цепь содержит четыре атома углерода.

CH₃–CH(CH₃)–CH₂–CH₃

2-метилбутан: заместитель CH₃ находится у второго атома.

Карта реакций

Что может происходить с алканом

Горение

CₙH₂ₙ₊₂ + (3n+1)/2 O₂ → nCO₂ + (n+1)H₂O

Уравнивайте в порядке C → H → O. Дробный коэффициент затем можно устранить умножением на 2.

Галогенирование

CH₄ + Cl₂ → CH₃Cl + HCl (hν)

Радикальное замещение идёт при освещении или нагревании и может продолжаться ступенчато.

Нитрование

RH + HNO₃ → RNO₂ + H₂O (t)

В реакции Коновалова H замещается нитрогруппой NO₂.

Сульфирование

RH + HOSO₃H → RSO₃H + H₂O

Это обобщённая школьная схема. Низшие алканы малоактивны; для высших нужны жёсткие условия, а промышленно используют и другие процессы.

Дегидрирование

C₄H₁₀ → C₄H₈ + H₂ (катализатор, t)

Удаление H₂ создаёт двойную связь; возможны разные положения связи.

Крекинг

C₅H₁₂ → C₂H₄ + C₃H₈ (t)

Цепь расщепляется на более короткие молекулы. Проверяйте баланс и четыре связи у C.

Термическое разложение метана

Около 1000 °C
CH₄ → C + 2H₂
Около 1500 °C
2CH₄ → C₂H₂ + 3H₂

Селективность галогенирования

В разветвлённых алканах замещение преимущественно идёт у более замещённого атома углерода: третичный H реакционноспособнее вторичного, вторичный — первичного. Но образуется смесь изомеров; правило указывает основной, а не единственный продукт.

Типичные ошибки

Пять связей у C.
Пересчитайте кратности всех связей.
Нет условий.
Для хлорирования нужен свет или нагревание.
Потеря H при крекинге.
Сверьте число каждого элемента слева и справа.
Один продукт.
Преимущественный изомер не исключает смесь.
Кислород первым.
В горении сначала уравнивайте C и H.
Одна CH₄ → C₂H₂.
Для ацетилена нужны две молекулы метана.

Итоговый чек-лист

  • Общая формула соответствует классу вещества.
  • У каждого C ровно четыре связи.
  • Условия реакции указаны над стрелкой.
  • Сохранено число атомов C и H.
  • Главный продукт не назван единственным без основания.
  • Коэффициенты сокращены до наименьших целых.

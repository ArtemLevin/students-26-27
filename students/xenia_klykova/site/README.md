# Веб-пособия для Ксении Клыковой

Многостраничный автономный сайт по мотивам файлов из `students/xenia_klykova`.

## Структура

- `index.html` — общий навигатор.
- `applied-formulas.html` — прикладные формулы, единицы, тепловое расширение, прибыль, отбор корней.
- `ege-models.html` — экономика, кинематика, термодинамика, графическая интерпретация корней.
- `powers-physics.html` — вытекание воды, куб величины, четвёртая степень, научная запись.

## Как открыть

Откройте `index.html` локально в браузере. Внешние зависимости не используются.

## Реализовано

- встроенные CSS и JavaScript;
- MathML для формул;
- тёмная и светлая тема;
- sticky canvas-блок;
- клик по canvas открывает модальное окно;
- синхронизация параметров основной и модальной модели;
- зум, сброс масштаба, колесо мыши, перемещение схемы мышью;
- пошаговое раскрытие решения;
- переключение подробности;
- динамические подсказки;
- генератор похожей задачи;
- мини-квиз и самопроверка;
- печатная версия.

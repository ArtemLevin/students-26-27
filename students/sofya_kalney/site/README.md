# Веб-пособия для Софии Кальней

Многостраничный автономный сайт по мотивам двух LaTeX-пособий из `students/sofya_kalney`.

## Файлы

- `index.html` — общий навигатор.
- `motion-basic.html` — базовые задачи на движение: навстречу, задержка старта, вдогонку, туда-обратно.
- `motion-advanced.html` — разность времён, системы, круговая трасса, движение по воде.

## Как открыть

Откройте `index.html` локально в браузере. Внешние зависимости не используются.

## Что реализовано

- тёмная и светлая тема;
- MathML-формулы без raw LaTeX;
- canvas-визуализация;
- клик по canvas открывает синхронное модальное окно;
- зум, сброс, перетаскивание мышью и масштабирование колесом;
- пошаговое раскрытие решения;
- переключение подробности;
- динамические подсказки;
- генератор одной похожей задачи;
- мини-квиз и самопроверка;
- печатная версия.

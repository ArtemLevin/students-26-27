Главная идея

Два слоя решения

  • Сначала обеспечьте существование каждого логарифма.
  • Затем определите монотонность по основанию.
  • Преобразуйте логарифмы и сравните аргументы.
  • Пересеките алгебраическое решение с исходным ОДЗ.
Финальная формула: решение = преобразованное неравенство ∩ ОДЗ.

Область допустимых значений

Три обязательных условия

logaF(x) существует, если a>0, a≠1,F(x)>0
Аргумент

Всегда строго положителен. Ноль не допускается даже при нестрогом знаке исходного неравенства.

Несколько логарифмов

Положительным должен быть каждый исходный аргумент.

После объединения

Исчезнувшее из записи ограничение не исчезает из ОДЗ.

Граница

Точка, где аргумент равен нулю, всегда исключается.

Пример с двумя аргументами

log2(x2−x−6) <log2(x−2)

ОДЗ: x²−x−6>0 и x−2>0. После разложения получаем x>3.

Только после этого сравниваем аргументы и в конце пересекаем результат с x>3.

Монотонность

Когда знак сохраняется

a > 1

Логарифмическая функция возрастает. Знак между аргументами сохраняется.

0 < a < 1

Функция убывает. Знак между аргументами меняется на противоположный.

logaF>logaG , при a>1⇒F>G
logaF>logaG , при 0<a<1⇒F<G

Разобранный пример

Для неравенства log₂(x²−x−6)<log₂(x−2) основание больше единицы:

x2−x−6 <x−2

Получаем x²−2x−4<0, то есть x∈(1−√5;1+√5). После пересечения с ОДЗ x>3:

Ответ: x∈(3;1+√5).

Основание меньше единицы

В примере с основанием 1/3 знак меняется. После объединения правой части:

log13(x2−3x−4) > log13(x2−1)

Сравниваем аргументы в обратную сторону. ОДЗ даёт x>4; итоговый ответ — (4;+∞).

Преобразования

Сумма, разность и константа

logau+logav =loga(uv)
logau−logav =logauv
1=logaa
Не путайте: logₐu + logₐv — это logₐ(uv), а не logₐ(u+v).

Пример: единица под логарифмом

1+log2x >log25

ОДЗ: x>0. Заменяем 1 на log₂2 и объединяем:

log2(2x) >log25
Ответ: x>5/2.

Замена переменной

Квадратное неравенство относительно логарифма

(log2x)2 −log2x−6≤0
ОДЗ: x>0.
Вводим t=log₂x.
Решаем (t−3)(t+2)≤0.
Получаем −2≤t≤3.
Возвращаемся: 2⁻²≤x≤2³.
Ответ: x∈[1/4;8].

Правило обратного перехода

При a>1 порядок границ сохраняется. При 0<a<1 порядок меняется, потому что функция убывает.

Важно: решение по t — ещё не ответ. Его нужно перевести обратно к x.
